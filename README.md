# Maps Navigation Web App

A Google Maps-style navigation web application built using Leaflet and OpenStreetMap.

## Features
- Location search with suggestions
- Current location detection
- Route navigation (start → destination)
- Distance and travel time calculation
- Nearby places discovery
- Live navigation simulation
- Mobile-style UI

## Tech Stack
- HTML
- CSS
- JavaScript
- Leaflet.js
- OpenStreetMap API

## How to Run
1. Download the project
2. Open index.html in your browser
